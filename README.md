# WDD 430 Portfolio

My first Next.js application, built for WDD 430 (Web Full-Stack Development) at BYU-Pathway.

## Live Site

**Deployed on Vercel:** https://wdd430-portfolio-ochre.vercel.app/

## About the Project

A simple portfolio site built with the Next.js App Router. It includes:

- A Home page (`/`) that displays my recent projects using reusable components
- An About page (`/about`) with my background and technical skills
- An API route (`/api/hello`) that returns a JSON response

## Tech Stack

- Next.js (App Router)
- React (Server Components)
- TypeScript
- Tailwind CSS

## Running Locally

```bash
yarn install
yarn dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
